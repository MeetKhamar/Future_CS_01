# Cyber Security Internship – Future Interns

This repository contains the work completed as part of the **Cyber Security Internship Program** at **Future Interns**.  
The tasks focus on practical security analysis, awareness, and reporting using ethical and non-intrusive methods.

---

## 📂 Tasks Included

### 🔹 Task 1: Vulnerability Assessment
- Passive security assessment of a public website
- Identification of common web security issues
- Risk classification (Low / Medium / High)
- Remediation suggestions
- Professional vulnerability assessment report with supporting evidence
